Cómo va, les comparto un pequeño proyecto de prueba, en resumen se trata de mostrar transferencias entre usuarios registrados en la app (asociación de username con  wallets verificadas).
La idea es mostrar transferencias que tienen una naturaleza pública, como entre organizaciones, y mostrarlas para aprovechar lo mencionado y generar confianza en su mercado.
Ejemplo, un sponsor transfiere a un club por un acuerdo de promoción.

Los datos que se obtienen desde la blockchain de Ethereum (api de alchemy)
Se hace una validación genérica de la wallet externa del usuario (envío de tokens a una dirección de la app a la cual se le está "escuchando").

El mayor desafio es el registro historico en la blockchain y la coordinación entre las wallets que el usuario valida. Más es algo que se puede lograr.
